Dot Network Variations

Animated graph explorations — speed, density, layout, color, glow, pulse, rotation & more

It started with a static image I found on Pinterest — a simple triptych showing dots on a circle, then lines drawn to the center forming a star, then every node connected to every other node in a fully-connected graph.

The original image that started it all
↓

I gave this image to Gemini and asked it to recreate the concept as a single animated SVG — dots that start alone, draw lines to the center, then gradually connect to each other in a fully-linked network before fading back to just dots again, looping forever.

Gemini's output — the animated SVG it generated from the image above
↓

Then I wondered: what if I changed the speed? The number of dots? The layout? The colors? This page is the result — dozens of variations exploring every axis of that one simple idea.
